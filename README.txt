##This is not under development and was tested on ROS Electric Emys. 
http://www.Playwithrobots.com 
sparkV_android is a basic ROS package that subscribes to imu messages published by phone and according to the phone's orientation publishes motion command to sparkV robot. 

Using Instruction at: http://playwithrobots.com/controlling-sparkv-using-android-phone-accelerometer
